Configuring the Olympus E-P1 for maximum responsiveness

Fig. 1: Grab shot taken using method described below (Pickering Nuclear Reactor in the distance)

This is the configuration I'm currently using for street/candids/snapshots work with raw output. For this type of photography I find I need instant response from the camera for at least a single shot at a time. Since I shoot raw and routinely post-process, I set exposure via the ETTR methodology (keep histogram to the right but without blowing non-specular highlights). Because pattern (ESP), centre-weighted, and standard spot metering options all work toward putting the "subject" at middle EV, they are not directly useful for ETTR. Highlight spot would be useful if other camera controls allowed.

For this sort of work both the E-P1's auto and manual focus systems are too slow to routinely re-focus before each shot, It's therefore imperative if using AF to disable the default expose+focus cycle attached to the shutter release button. The only way I've found to do this is to put the camera in manual focus mode, assign single-shot AF to the AEL/AFL button, and press it whenever I need to re-focus. (Alternately, I could do an actual manual focus via the magnifier feature, but I find this takes me longer than a single AF event.)

The above would be acceptable if I could trust any of the available automatic exposure methods, but my insistence on not blowing highlights prevents me from doing that. And since I've already assigned S-AF to the AEL/AFL button, I no longer have an important tool in the exposure arsenal.

When working in lighting that remains fairly stable from moment to moment, I generally use M mode with pattern metering (ESP) and on-screen histogram, change f/stop with the upper dial (infrequent), and change shutter with the lower dial since it spins so easily. As I move from one lighting situation into another, I re-dial the shutter speed while eyeing the histogram to keep the brightest values just at the right edge.

The above is workable in many cases, but when I have a subject that keeps moving between different lighting situations, such as from sunlight to shade, or the light itself keeps changing, this approach becomes impractical. Then I switch to Av mode, grab an acceptable exposure while glancing at the histogram, by panning about the frame then half-pressing the shutter release when the histogram looks good, then keep the shutter release half-pressed until I've got the shots I want in that light.

If there isn't even time to grab an exposure, I'm left with little choice but to use Av mode and simply accept the (often disastrous!) recommendation of the ESP pattern.

Below are the settings I use to do the above. Any menu options I don't record here are irrelevant:

Mode dial:

  • M, A, or S

Live View options:

  • WB: Auto (but pre-setting a balance improves performance)
  • Shutter mode: Sequential (I'll have more to say about this soon)
  • IS: 1
  • Output: I use RAW or Large Fine + RAW
  • Meter: ESP
  • Focus: Manual
  • ISO: (as appropriate but not auto, since I want to be able to operate at low shutter speeds)
  • Face detect: Off
  • Focus area: (single centre point)

Set-up menu:

  • Rec View: Off
  • LCD brightness: +7 for bright daylight; 0 indoors
  • Cog menu display: On

Cog menu:

A:

  • MF Assist: On

B:

  • Dial Function: Av - top dial Fno, bottom +/-
  • Dial Function: M - top dial Fno, bottom Shutter
  • AEL/AFL: S1/C2/M3
  • AEL/AFL Memo: On
  • Fn Function: Backlit LCD
  • Left arrow Function: Metering
  • My Mode Setup: (unused)
  • Button Timer: 3 sec
  • AEL/AFL<->Fn: Off
  • 4-Way Function: On

E:

  • EV Step: 1/3
  • AELMetering: Auto
  • ISO Step: 1 EV

That's how I set it up; actually putting it to use takes quite a bit of practice, so don't expect everything to instantly fall into place. I'll be adding to this page in days to come.
