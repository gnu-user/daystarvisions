The following is my current Photoshop procedure for making JPEGs. Improvements and corrections very much solicited.

  • If you use a Mac and have your monitor gamut set to 1.8, temporarily change it to 2.2. Only a dwindling number of Mac users use 1.8, and your profile will automatically convert down to 1.8 for them.
  • Open final edit of image to be copied.
  • Image/Duplicate it and work with duplicate.
  • If multiple layers, reduce to one; if alpha channels, delete them.
  • Image/Image Size down to desired jpeg size.

Most people use 1024x768 screen res with maybe 5% larger and a dwindling minority of 800x600. 800x600 is the size I normally use for gallery/album images - fills most of a 1024x768 display and all of a 800x600 display. (Actually, I set the length to 776 to leave room for a border.) Images meant for illustrations should usually be smaller - perhaps 400x300. Images in portrait orientation should be 600 pixels or shorter in height.

  • Convert to profile sRGB for best colour preservation.

Assuming your image is in Adobe RGB. sRGB is similar to the gamut of monitors. Most people use Windows, which doesn't support profiles in browsers, so the profile gets stripped away. Adobe RGB gamut is severely clipped if stripped away, but sRGB gamut is not.

I find reducing my print-sharpened 6000x4000 pixel images down to 800x600 leaves them soft:

  • Apply mild edge sharpening, or regular USM as in next three steps:
  • Create a duplicate layer.
  • Sharpen it with mild USM. Amount = 50; Radius = 1.5; Threshold = 5 is my starting point.
  • If there are offensive haloes, use eraser at low strength to bleed in the unsharpened background layer, then merge layers.
  • Use File/Save for Web... to convert to JPEG. Settings: JPEG, quality around 60 (whatever it takes to get the file size down to your target), ICC Profile checked, save as type Image Only. Leave everything else at defaults.

My file size targets are 200 KB for album/gallery pix, and roughly 75 KB for illustrations. The more compression you apply the more your carefully achieved sharpening will be ravaged, so trade-offs always involve some angst.

If you want to add a simple border:

  • Set the Photoshop background colour to whatever you want for the border.
  • Open Image/Canvas Size..., set the width and height to some larger number of pixels.
